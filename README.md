# PROJETOS PYTHON

Este repositório contém os meus primeiros projetos de estudo em Python, focados em automação e cibersegurança.

## PROJETOS INCLUÍDOS:

### 1. Conversor de Moedas
- **Descrição:** Script para conversão de valores entre diferentes moedas (Real/Dólar/Iene).
- **Funcionalidades:** Utiliza lógica de programação e dados e cálculos matemáticos básicos.

### 2. Scanner de Portas
- **Descrição:** Ferramenta simples para verificar portas abertas em um IP alvo.
-**Funcionalidades:** Utiliza a biblioteca 'socket' para testes de conexão.

## Como executar
1. Certifique-se de ter o Python instalado.
2. Clone o repositório:
    'git clone https://github.com/renatomatsuoo/projetos-python.git'
3. Execute o script desejado:
    'python scanner_portas.py'
